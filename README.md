# Weclome 

Hi and welcome to the main gui for Internal Inventory Tracker

to get started please ensure that you

1. Have Internal Inventory Tracker Go APi on and running on your machine 

2. Ensure you have flutter installed on your machine 

3. Ensure that you have Ports 5000 to 6000 are not currently in use by any other program on your system 


one that is all sorted you may fine the following usefull

Clean run

flutter clean - this will clear out any previous dependencies

flutter pub get - this will get back dependencies needed for my gui to run

flutter run -d chrome - this will run the gui in a chrome tab
